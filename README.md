# This project is a two part project

## To compile the solidity code

### node compile.js

## To deploy the solidity code

You need to set two environment variables: `INFURA_ROPSTEN_ENDPOINT` and `MNEMONIC_PHRASE_WALLET` and run:

### node deploy.js
